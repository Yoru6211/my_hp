# my_hp

撮影した写真を公開するためのホームページです。<br>
http://photogallery.renhattori.com/

## 概要
撮影した写真に加えて、写真の情報（撮影機種、焦点距離、撮影モード、絞り、シャッタースピード、ISO感度、露出補正、ホワイトバランス）も合わせて
公開しています。
